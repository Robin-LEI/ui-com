# 按钮

```html

<template>
  <ed-button>默认按钮</ed-button>
  <ed-button type="login">登录按钮</ed-button>
  <ed-button type="primary">主要按钮</ed-button>
</template>
<!-- live-0-0.vue -->
```

### Attributes

| 参数 | 说明 | 类型 | 默认值 |
| ---- | ---- | ---- | ------ |
| type | 按钮类型 | string | 默认 |
| click | 绑定事件 | - | - |
